# GitHub Desktop을 사용한 업로드 가이드

## 📥 Step 1: GitHub Desktop 설치

1. 브라우저에서 열기:
   https://desktop.github.com

2. "Download for Windows" 클릭

3. 다운로드한 파일 실행 (GitHubDesktopSetup.exe)

4. 자동 설치 진행 (1-2분)

5. GitHub Desktop 실행

---

## 🔐 Step 2: GitHub 로그인

1. "Sign in to GitHub.com" 클릭

2. 브라우저가 열리면 로그인

3. "Authorize desktop" 클릭

4. GitHub Desktop으로 돌아옴

---

## 📁 Step 3: 기존 저장소 추가

1. GitHub Desktop 메뉴:
   File → Add local repository

2. "Choose..." 버튼 클릭

3. 폴더 선택:
   F:\06_일본미스테리\AutoGen_Pro

4. "Add repository" 클릭

---

## 🚀 Step 4: 저장소 퍼블리시

1. 상단에 "Publish repository" 버튼 보임

2. 클릭하면 나오는 창에서:
   - Name: Japan-Mystery (자동 입력됨)
   - Description: 일본 미스테리 Shorts 자동 생성 플랫폼
   - ☐ Keep this code private (체크 해제 - Public으로)
   또는
   - ☑ Keep this code private (팀원만 접근)

3. "Publish repository" 버튼 클릭

4. 업로드 진행 (1-2분)

---

## ✅ Step 5: 확인

1. 브라우저에서 확인:
   https://github.com/leestar20251214-sys/Japan-Mystery

2. 28개 파일 확인:
   - README.md
   - QUICK_START.md
   - App.tsx
   - components/
   - services/
   - 등등...

---

## 🎉 완료!

이제 팀원들을 초대할 수 있습니다!

Settings → Collaborators → Add people

## 💡 앞으로 사용법

### 코드 수정 후 업로드:

1. GitHub Desktop 열기
2. 왼쪽에서 변경된 파일 확인
3. 하단 "Summary" 입력:
   예: "기능 추가: 자동 저장"
4. "Commit to main" 클릭
5. 상단 "Push origin" 클릭
6. 완료!

**매우 쉽습니다!** 클릭 몇 번으로 끝!
